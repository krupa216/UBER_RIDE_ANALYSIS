**Uber Ride Analysis**

**Project Overview**

This project performs Exploratory Data Analysis (EDA) on an Uber Trips dataset to understand ride patterns, customer payment preferences, and trip characteristics. The analysis focuses on identifying trends in trip demand, fare distribution, and ride distance using data visualization techniques.

**Dataset**

The project uses the Uber Trips Dataset 50K, which contains information about trip details, locations, payment methods, fares, and timestamps.

**Key Features**
City
Pickup and Drop Location
Distance (km)
Fare Amount
Payment Method
Trip Status
Pickup Time and Drop Time

**Tools and Technologies**
Python (Google Colab) for data analysis
Pandas, NumPy for data manipulation
Matplotlib, Seaborn for data visualization
Power BI for interactive dashboard creation
Analysis Performed
Trip distribution by city
Payment method analysis
Distance and fare distribution
Trip demand by hour
Correlation analysis between numerical variables

**Key Insights**
Some cities generate higher ride demand than others.
Most rides are short-distance trips.
Digital payment methods are commonly used.
Trip demand peaks during commuting hours.
